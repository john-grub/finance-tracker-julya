HappyTracker — Personal
Finance Tracker

A production-grade budget management and finance tracking web app. Users can create custom budget categories, track spending, set financial goals, and get insights from a live dashboard.

✦ Core Features
📊

Budget Dashboard

Live overview of total income, expenses, savings rate, and spending by category with visual charts.

🗂️

Custom Budget Categories

Users create, name, color-code, and set spending limits for their own budget categories (Food, Travel, etc.).

🎯

Budget Goals

Set savings or spending goals with deadlines. Track progress with visual indicators and alerts.

💸

Transaction Logging

Add, edit, and delete transactions. Assign to any custom category. Filter by date, type, or category.

🔐

Auth & Multi-User

Secure JWT-based login and registration. Each user's data is fully isolated and private.

📅

Monthly Reports

View spending summaries by month. Compare months side by side to spot patterns.

✦ 2-Week Dev Timeline
📌 Week 1 — Planning, architecture, and backend foundations. Database, auth, and core APIs are the priority.
Planning Day 1–2 — Project Setup & Architecture Mon – Tue ▼
Backend
Initialize Spring Boot Project
Create Maven/Gradle project. Add Spring Web, Spring Data JPA, Spring Security, JWT, and PostgreSQL dependencies.
Database
Set Up PostgreSQL & Schema
Create local DB. Write schema SQL for users, budget_categories, transactions, budget_goals tables.
Frontend
Initialize React App
Bootstrap with Vite. Install React Router v6, Axios, Recharts, Tailwind or CSS modules.
DevOps
Git Repo & Folder Structure
Initialize monorepo. Set up .env for secrets. Define project folder conventions for both backend and frontend.
Planning Day 3–4 — Auth & User Management Wed – Thu ▼
Backend
JWT Auth Endpoints
Build POST /api/auth/register and POST /api/auth/login with password hashing (BCrypt).
Backend
Spring Security Config
Configure JWT filter, SecurityFilterChain, and per-request user extraction middleware.
Frontend
Login & Register Pages
Build auth forms with React Router. Store JWT in localStorage. Create AuthContext with useContext for global auth state.
Frontend
Protected Routes
Create PrivateRoute component wrapping React Router's Outlet. Redirect unauthenticated users to /login.
Build Day 5–7 — Budget Categories & Transactions API Fri – Sun ▼
Backend
Budget Categories CRUD
REST endpoints: GET, POST, PUT, DELETE for /api/categories. All scoped to the authenticated user.
Backend
Transactions CRUD
Full CRUD for /api/transactions. Support filtering by month, year, and category_id query params.
Frontend
Budget Categories UI
Create categories page with a card grid. Add/Edit/Delete forms. Color-picker and emoji icon support.
Frontend
Transactions Page
Transaction table with filters. Add transaction modal with category dropdown and date picker.
📌 Week 2 — Dashboard, goals, frontend polish, and testing/deployment.
Build Day 8–9 — Dashboard & Goals Mon – Tue ▼
Backend
Dashboard Summary Endpoint
Build GET /api/dashboard/summary returning total income, total expenses, savings, and per-category breakdown.
Backend
Goals API
CRUD for /api/goals. Include computed progress_percentage field. Endpoint to update current_amount.
Frontend
Dashboard Page
Stat cards (income, expenses, savings). Recharts donut for category spending. Recent transactions list.
Frontend
Goals Page
Goal cards with progress bars, deadlines, and status chips. Create/edit goal modal.
Build Day 10–11 — Frontend Polish & UX Wed – Thu ▼
Frontend
Responsive Layout
Ensure sidebar collapses on mobile. Responsive stat grid and table. Hamburger menu.
Frontend
Loading & Error States
Skeleton loaders for all async data. Global Axios error interceptor with toast notifications.
Frontend
Monthly Report View
Month selector with bar chart comparing income vs expenses. Per-category breakdown table.
Backend
Budget Limit Alerts
Add logic to flag categories that exceed their monthly_limit. Surface warning in dashboard API response.
QA Day 12–13 — Testing & Bug Fixes Fri – Sat ▼
Backend
Unit Tests (JUnit + Mockito)
Write unit tests for service layer: CategoryService, TransactionService, GoalService, AuthService.
Backend
Integration Tests
Test REST controllers with MockMvc. Test auth flow, CRUD operations, and edge cases (empty data, invalid JWT).
Frontend
E2E Manual QA
Walk through full user journey: register → create category → add transactions → set goal → view dashboard.
Both
Bug Fixes
Fix identified issues, CORS config, edge cases for empty states, date timezone handling.
Deploy Day 14 — Deployment & Wrap-Up Sunday ▼
DevOps
Backend Deployment
Package Spring Boot as JAR. Deploy to Railway or Render. Set PostgreSQL env vars. Verify API health endpoint.
DevOps
Frontend Deployment
Build React app with Vite. Deploy to Vercel or Netlify. Set VITE_API_BASE_URL to production backend URL.
Docs
README & Documentation
Write README with setup instructions, env variables, and API overview. Add Swagger/OpenAPI docs to Spring Boot.
Review
Final Review
Demo walkthrough. Verify all features work in production. Tag release v1.0.0 in Git.
